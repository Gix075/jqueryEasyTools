googleReCaptcha Validator

Overview

This tool is a validator for googleReCaptcha security system. It works with javascript and php and it's very simple to use.
See below and usage example and read documentation to learn how integrate this tool on your site.

How it Works

In order to install and use this tool, register your domain at https://www.google.com/recaptcha/ and generate pubblic and private keys.

Download & Install

Download googleReCaptcha and uncopress it on your local machine. You will have the following file tree:

  • js/reCaptchaValidator.js
  • php/reCaptchaValidator.php
  • demo.html

demo.html is a demo webpage that you can use as a test page to learn tool usage.
Note that the demo page will not work without a correct configuration.
